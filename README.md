# learnwith
We will use this repo as an issue tracker for our [Learnwith site](https://learnwith.weareopen.coop). Learn more about We Are Open Co-op at [our website](https://weareopen.coop). This repo also has a wiki to explain how to create content for the Learn with WAO platform, make changes and get some technical background to some of the plugins that have been used.
